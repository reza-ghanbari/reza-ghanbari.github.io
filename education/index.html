Research Experience


Java String Verification by JayHorn repository

  • Under supervision of Prof. Hosein Hojjat
  • This research aimed to define models for String methods in Java by means of Constrained Horn Clauses to statically verify programs containing those methods. The models have been implemented in JayHorn Framwork, an unbounded model checker for Java, and the tool is going to compete with other verifiers in SV-COMP 2022.

Real Time Verification and Evaluation for Rebeca Language repository

  • Under supervision of Prof. Fatemeh Ghassemi
  • Rebeca is an actor-based language used for formal verification. In this research, a tool has been created that runs Rebeca code and evaluates its correctness based on a run-time enforcement algorithm.

Decentralized Enforcement in the Orbit Framework repository

  • Under supervision of Prof. Fatemeh Ghassemi
  • This research has been conducted to enhace the orbit framework, an actor based system, to identify and prevent some sequence of messages which may violate system functionalities. To achieve this, a decentralized algorithm has been desinged and is being implemented, and also the framework has changed to adapt new algorithm.

Rebeca Code Generator repository

  • Under supervision of Prof. Fatemeh Ghassemi
  • In this research, a configurable code generation tool for rebeca framework has been developed that is used in order to test some functionalities of rebeca compiler such as deadlock detection.
